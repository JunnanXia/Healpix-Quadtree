# WHI paper support materials

This folder contains supplementary scripts and supporting materials related to the WHI paper.

These contents are not part of the HPQT core algorithm itself.  
They are included only as auxiliary materials for selected components of the broader WHI workflow used in the study.  
For methodological details, please refer to the paper.  
The WHI datasets for 2000, 2005, 2010, 2015, and 2020 are available at: https://zenodo.org/records/19112122
